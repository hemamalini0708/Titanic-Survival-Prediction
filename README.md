# **Titanic Survival Prediction**

**DEMO :- https://www.loom.com/share/e9410e7281bf46b2a911a541dfee7f41?sid=89207d81-79d2-4a90-bbaa-4ec9d9c0f1df**    

**Content:**

The Titanic Survival Prediction System allows users to input key passenger information and choose a machine learning algorithm to predict whether the passenger would have survived the Titanic disaster.

**How It Works:**

**User Input:**
Age, Parents/Children Aboard, Fare (£), Sex (Male/Female), Port of Embarkation (C, Q, S)


**Select Algorithm:**

Users can choose from the following machine learning models to make the prediction: Naive Bayes, Logistic Regression, Decision Tree


**Analysis:**
The system processes the input using a machine learning model trained on historical Titanic passenger data to identify survival patterns.


**Prediction:**
After submitting the form, the system processes the inputs and displays whether the passenger would have Survived or Not Survived based on the selected algorithm.
